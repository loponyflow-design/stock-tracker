# NBIX — Neurocrine Biosciences
_Brief date: 2026-07-02_

> ⚠️ รายได้เกือบ 80% กระจุกอยู่ที่ยาตัวเดียว (INGREZZA) และ patent cliff เริ่ม 2027 — อ่าน Kill conditions เป็นพิเศษก่อนถือระยะยาว

---

## 1. Company snapshot

Neurocrine เป็นบริษัทยา neuroscience ที่ขายยาให้แพทย์ระบบประสาท/จิตเวชสั่งจ่ายให้ผู้ป่วย รายได้เกือบทั้งหมดมาจากยา INGREZZA (รักษาอาการเคลื่อนไหวผิดปกติจากยาจิตเวช/tardive dyskinesia และ Huntington's disease) ซึ่งเป็นยาตัวหลักที่ทำรายได้กว่า 80% ของบริษัท ยาตัวใหม่ CRENESSITY (รักษา congenital adrenal hyperplasia) เพิ่งเปิดตัวปลายปี 2024 กำลังเติบโตเพื่อกระจายความเสี่ยง

---

## 2. Fundamentals signal

- **โตเร็วต่อเนื่อง 3 ปี:** รายได้ $1,887M → $2,355M → $2,861M (CAGR ~23%) แต่การเติบโตของ INGREZZA เองเริ่มชะลอ (จาก +26% เหลือ +8.6% ปีล่าสุด) ส่วนที่เพิ่มมาจาก CRENESSITY เกือบทั้งหมด
- **Gross margin สูงมากแบบธุรกิจยา:** 97.9-98.6% ต่อเนื่อง — ต้นทุนการผลิตต่ำเมื่อเทียบราคาขาย เป็นลักษณะทั่วไปของ specialty pharma ที่จดสิทธิบัตรคุ้มครอง
- **Operating margin หดตัวจาก R&D ที่ทุ่มเพิ่ม:** จาก 24.5% เหลือ 21.6% เพราะ R&D โต 39% เพื่อสร้าง pipeline รองรับหลัง INGREZZA หมด exclusivity — เป็นการลงทุนที่จำเป็นมากกว่าสัญญาณแย่
- **งบดุลไม่มีหนี้ เงินสดเพิ่มทุกปี** เป็น $2.5B — มีกระสุนพอสำหรับซื้อกิจการเสริม pipeline (ล่าสุดตกลงซื้อ Soleno Therapeutics ~$2.9B พ.ค. 2026)
- **จัดสรรทุนเน้น R&D และซื้อกิจการมากกว่าคืนผู้ถือหุ้น** — ไม่มีปันผล ซื้อหุ้นคืนลดลงจากปีก่อน สะท้อนว่าฝ่ายบริหารเลือกลงทุนอนาคตมากกว่าจ่ายคืนตอนนี้

## 4. Bull case / Bear case

**Bull**
- **Cornered resource ชั่วคราวจากสิทธิบัตร + orphan drug exclusivity** — INGREZZA และ CRENESSITY มีการคุ้มครองสิทธิบัตร/exclusivity ที่ทำให้คู่แข่ง generic เข้าตลาดไม่ได้ในช่วงเวลาหนึ่ง (CRENESSITY คุ้มครองถึงอย่างน้อยปี 2031)
- **กระแสเงินสดสูงมากพอจะซื้อ pipeline ใหม่ต่อเนื่อง** — ไม่มีหนี้ เงินสด $2.5B ทำให้ซื้อกิจการเสริม (Soleno) ได้โดยไม่ต้องกู้หนักหรือ dilute ผู้ถือหุ้นมาก
- **Pipeline หลายตัวอยู่ระหว่าง Phase 2-3** (osavampator, direclidine, NBI-1117570) — ถ้าสำเร็จจะช่วยกระจายรายได้ก่อนที่ INGREZZA จะเจอ patent cliff

**Bear**
- **รายได้กระจุกตัวที่ยาตัวเดียวสูงมาก** — INGREZZA เป็น 81% ของยอดขาย ถ้ามีปัญหาด้านความปลอดภัยยา (safety signal) หรือคู่แข่งออกยาใหม่ที่ดีกว่า รายได้บริษัทจะกระทบทันที
- **Patent cliff ของ INGREZZA เริ่มปี 2027** — orphan drug exclusivity สำหรับ tardive dyskinesia หมดไปแล้ว เหลือ exclusivity เฉพาะ Huntington's disease ถึงปี 2030 เท่านั้น หลังจากนั้นความเสี่ยง generic เข้าตลาดจะเพิ่มขึ้นชัดเจน
- **Pipeline ที่จะมาทดแทนยังอยู่ระยะ Phase 2-3** — ยังไม่มีข้อมูลยืนยันว่าจะสำเร็จ ธุรกิจยาแบบนี้ risk สูงกว่าธุรกิจ consumer/industrial ทั่วไปเพราะผลการทดลองอาจล้มเหลวได้ทุกเมื่อ

---

## 5. Kill conditions

- **ถ้า INGREZZA เริ่มเห็นยอดขายลดลงจริงจากการแข่งขัน generic หรือคู่แข่งใหม่** (ไม่ใช่แค่ growth ชะลอ) — เพราะเป็นรายได้หลักเกือบทั้งหมด
- **ถ้า pipeline หลัก (osavampator, direclidine) ล้มเหลวใน Phase 3** — จะทำให้บริษัทพึ่งพา INGREZZA ต่อไปโดยไม่มีตัวทดแทนชัดเจนก่อนถึง patent cliff
- **ถ้ามีปัญหาด้านความปลอดภัย (safety/FDA label warning) กับ INGREZZA** — ยาตัวเดียวที่เป็นเสาหลักรายได้ กระทบได้แรงกว่าธุรกิจที่กระจายสินค้า

## 6. What to ask before owning it

1. ผมยอมรับความเสี่ยงที่รายได้ 80%+ ขึ้นกับยาตัวเดียวได้แค่ไหน เทียบกับธุรกิจที่กระจายสินค้า/ลูกค้า?
2. Patent cliff ของ INGREZZA ปี 2027 จะกระทบรายได้เร็วแค่ไหน และบริษัทมีแผนสำรองอะไรนอกจาก CRENESSITY?
3. การซื้อ Soleno Therapeutics มูลค่า $2.9B คุ้มค่าจริงหรือเป็นการซื้อการเติบโตด้วยราคาแพง?
4. Pipeline ที่อยู่ระหว่าง Phase 2-3 มีโอกาสสำเร็จตามสถิติอุตสาหกรรม (base rate) เท่าไหร่ ผมพร้อมรับความไม่แน่นอนนี้ไหม?
5. "เงื่อนไขที่จะทำให้ผมขายทิ้ง" คืออะไร — ยอดขาย INGREZZA เริ่มหดตัวจริง, pipeline ล้มเหลว, หรือ CRENESSITY โตไม่ทันชดเชย?
