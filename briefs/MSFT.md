# Microsoft Corporation (NASDAQ: MSFT)
_brief วันที่ 16 มิ.ย. 2026 | source: 10-K FY2025 + Web search (Q3 FY2026 results)_

---

## 1. Company Snapshot

Microsoft เป็นบริษัทซอฟต์แวร์และ cloud infrastructure ที่ขายให้กับองค์กรและผู้บริโภคทั่วโลก รายได้หลักมาจาก 3 กลุ่ม: Microsoft 365 + LinkedIn + Dynamics (43%), Azure cloud (38%) และ Windows + Xbox + Bing (19%) จุดแข็งที่แท้จริงคือ Switching Costs ระดับสูงมาก — องค์กรที่ใช้ Windows → M365 → Teams → Dynamics → Azure แทบไม่มีทางย้ายออกโดยไม่เจ็บปวด รายได้ 77% เป็น subscription/cloud ที่เก็บซ้ำทุกเดือน ทำให้ revenue predictable กว่าบริษัท tech ส่วนใหญ่ และพาร์ทเนอร์กับ OpenAI ทำให้ Azure เป็น delivery channel หลักของ AI workloads ในตลาด enterprise ณ ปัจจุบัน

## 2. Fundamentals Signal

- **Revenue เร่งตัวชัด**: $212B (FY2023) → $245B (FY2024) → $282B (FY2025) +15% YoY; Azure re-accelerated เป็น +34% FY2025 และล่าสุด Q3 FY2026 อยู่ที่ +40% — ทิศทางชี้ขึ้น ไม่ใช่ชะลอ
- **RPO $375B = 1.3x annual revenue ที่ contract แล้ว**: Remaining Performance Obligations (unearned + contracted not-yet-invoiced) อยู่ที่ $375B ณ มิ.ย. 2025 โดย ~40% ($150B) จะ recognize ใน 12 เดือนถัดไป — นี่คือ revenue visibility ที่ผิดปกติสูงมาก ช่วย buffer ถ้า new deal ชะลอ
- **Operating margin ขยายตัวต่อเนื่อง**: 41.8% (FY2023) → 44.6% (FY2024) → 45.6% (FY2025) เพราะ opex โตช้ากว่า revenue แต่ cloud gross margin บีบตัวเล็กน้อย (69.8% → 68.8%) จาก AI datacenter depreciation — depreciation เพิ่ม 45% YoY ตาม capex ต้องติดตาม
- **งบดุลแข็งแต่ obligation ซ่อนอยู่**: cash $94.6B vs long-term debt $43B = net cash สุทธิ แต่ total contractual obligations รวม $397B (debt + construction + leases $178.7B + purchase commitments $110B) — operating lease ระยะยาวส่วนใหญ่คือ datacenter lease หลายทศวรรษ
- **FCF ถูก squeeze โดยตั้งใจ**: Operating CF $136B (+18%) แต่ capex $64.6B (+45%) กด FCF เหลือ $72B — capital return $42.5B/ปี ($18.4B buyback + $24.1B dividend) สม่ำเสมอ แต่เป็น priority รองจาก capex อย่างชัดเจนใน cycle นี้

## 4. Bull Case / Bear Case

### Bull
- **Switching Costs ระดับ enterprise ขั้นสูง**: องค์กรที่ใช้ M365 + Azure + Dynamics ต้องลงทุน implementation, training, และ data migration ถ้าจะย้าย — ต้นทุนการเปลี่ยน vendor สูงกว่าต้นทุน subscription ทุกปีหลายเท่า ทำให้ churn ต่ำมาก และการขาย Copilot add-on ง่ายกว่าหาลูกค้าใหม่
- **Cornered Resource ผ่าน OpenAI**: Microsoft เป็น exclusive Azure API provider + มี right of first refusal บน OpenAI capacity — หมายความว่า enterprise ที่ต้องการ GPT-4o/o1 ระดับ production ต้องผ่าน Azure เท่านั้น; Azure growth +40% Q3 FY2026 บอกว่าลูกค้าจริงจ่ายเงินแล้ว
- **Copilot upsell เป็น revenue engine ใหม่ที่ไม่ต้องหาลูกค้าใหม่**: 20M paid seats จาก base M365 Commercial ที่มี ~400M seats — penetration rate ยังต่ำ แต่ถ้าถึง 10% ของ 400M seats = 40M seats × $30 ARPU = $14.4B ARR เพิ่มเติมที่ cost of acquisition เกือบ $0

### Bear
- **Capex bet ขนาดใหญ่ + payback period ไม่ชัด**: FY2025 capex $64.6B + construction commitments $32.1B + purchase commitments $109.9B — ถ้า Azure AI demand ไม่โตเร็วพอ ค่าเสื่อม datacenter จะกด gross margin และ FCF ต่อเนื่องหลายปี; YTD -17% สะท้อนว่าตลาดยังไม่เชื่อ timeline ROI
- **Cloud gross margin structural pressure**: AI infra depreciation กด Microsoft Cloud gross margin จาก 71% → 69% FY2025 — GPU/memory costs ยังสูง และ pricing power ของ Azure ถูกแข่งกับ AWS + GCP ถ้า margin ลงต่อกว่า 65% รูปแบบธุรกิจ cloud จะเปลี่ยนไปมาก
- **OpenAI dependency risk**: ถ้า OpenAI ขยาย distribution ไป non-Azure (ผ่าน Amazon, Google หรือ direct API) โดยหา loophole ใน exclusive agreement — Cornered Resource ที่เป็น bull case หลักจะหายไปทันที; governance ของ OpenAI เป็น black box จากภายนอก

## 5. Kill Conditions

เงื่อนไขที่ถ้าเกิดขึ้นควรทบทวนการถือ:

1. **Azure growth ชะลอต่ำกว่า 20% เป็น 2 ไตรมาสติดกัน** — แปลว่า AI workload conversion ไม่เกิดตามที่บริษัทพนัน capex ไปมหาศาล; capex ที่ลงไปจะไม่คืนทุนในเวลาที่ acceptable
2. **Microsoft Cloud gross margin ร่วงต่ำกว่า 65%** — จุดนี้แปลว่า AI infra cost structure บั่นทอน economics ของ cloud จริงๆ ไม่ใช่แค่ temporary; operating leverage ที่เคยดีจะหายไป
3. **OpenAI เปิด API บน AWS หรือ GCP โดยไม่มีข้อจำกัด** หรือมีข่าว Microsoft เสียสิทธิ์ exclusivity — ต้องรีประเมิน bull case ทันทีเพราะ competitive advantage หลักเปลี่ยนไปอย่างมีนัยสำคัญ

## 6. What to Ask Before Owning It

1. **FY2026 capex guided สูงมากขึ้น — ROI จะมาจากไหนและเมื่อไร?** Azure ต้องโตที่ระดับไหนเพื่อ justify capex ระดับนี้ และ payback period คือกี่ปี? ถ้าตอบไม่ได้ แปลว่าซื้อโดยหวังว่า bet จะถูกโดยไม่เข้าใจ mechanics

2. **M365 Copilot 20M seats — ARPU uplift คือเท่าไรต่อ seat จริงๆ?** ราคา list $30/user/month แต่ enterprise มักต่อรองส่วนลด; ถ้า effective ARPU ต่ำกว่ามาก revenue opportunity ที่คำนวณไว้จะเล็กลง

3. **OpenAI partnership — exclusive ในระดับใดและถึงเมื่อไร?** มีสัญญา exclusivity อยู่จนถึงปีไหน? Microsoft มีสิทธิ์ first refusal บน capacity แต่ถ้า OpenAI สร้าง capacity บน non-Azure cloud เงื่อนไขจะ trigger ไหม?

4. **MSFT down 17% YTD ในขณะที่ Azure +40% — discount ที่เห็นอยู่เป็น value หรือ warning?** ตลาดกำลังบอกอะไร — กังวลเรื่อง capex ROI, competition, หรือแค่ rotation ออก tech? ต้องมีมุมมองของตัวเองก่อนซื้อในช่วงที่คนอื่น exit

5. **ถ้า AI hype ลดลงใน 12-18 เดือน enterprise จะชะลอ Azure migration ไหม?** M365 subscription มี switching cost สูง แต่ Azure workload บางส่วน discretionary — ถ้า CFO ขององค์กรลูกค้าตัด cloud budget Microsoft จะ exposed มากน้อยแค่ไหน เทียบกับ on-premise alternative ที่มีอยู่

6. **RPO $375B ดูน่าเชื่อถือแค่ไหน?** $375B contracted revenue ฟังดูดี แต่ contract อาจ cancel หรือ reduce ได้; อยากรู้ว่า churn rate ของ RPO ในปีก่อนๆ เป็นเท่าไร และ Azure deal ที่นับเป็น RPO มี minimum commit จริงหรือแค่ commit-to-consume ที่ลูกค้าอาจไม่ใช้จนครบ
