# ARM — Arm Holdings plc
_Brief date: 2026-06-12_

---

## What the company does

ARM ออกแบบ CPU architecture และ licenses IP ให้บริษัทชิปทั่วโลก อย่าง Apple, Qualcomm, Samsung, MediaTek — โดยไม่ผลิตชิปเองเลย รายได้มาจาก 2 ทาง คือ license fees (จ่ายครั้งเดียวเมื่อได้รับสิทธิ์) และ royalties (จ่ายต่อชิปทุกตัวที่ ship ออกตลาด) ARM ครองตลาด mobile processor แทบ 100% ทั่วโลก และกำลังขยายสู่ data center, AI accelerator และ automotive

## Bull case / Bear case

**Bull**
- ARM มี **Switching Cost** สูงมาก — toolchain, developer knowledge และ IP ecosystem ที่สะสมหลายสิบปี ย้ายออกแปลว่าสร้าง ecosystem ใหม่ทั้งหมด
- v9 upgrade cycle ยังอยู่กลางทาง — ทุกครั้งที่ลูกค้าเปลี่ยนจาก v8 → v9 ARM ได้รายได้ต่อชิปสูงขึ้นโดยไม่ต้องหาลูกค้าใหม่
- Data center และ automotive เพิ่งเริ่ม — addressable market ที่ยังไม่ถูก monetize ยังมีอีกมาก

**Bear**
- RISC-V คือ open-source alternative ไม่มีค่า royalty — ถ้า ecosystem ครบพอ ลูกค้า IoT/embedded อาจไม่ต้องการ ARM อีก
- Apple และ Qualcomm รวมกันเป็นสัดส่วนใหญ่ของรายได้ — ถ้าฝ่ายใดต่อรองราคาสำเร็จหรือออกแบบทางเลือก กระทบหนัก
- Valuation สูงมาก สะท้อน perfect execution ไว้แล้ว ถ้า growth ชะลอแม้ชั่วคราว ราคามักถูกลงโทษรุนแรง

## What to ask before owning it

1. ผมเข้าใจ model ของ ARM ชัดไหม — ARM ไม่ขายชิป แต่เก็บเงินต่อชิปทุกตัวที่คนอื่นขาย ถ้า smartphone market flat ARM ยังโตได้ยังไง (hint: v9 royalty rate + data center)
2. RISC-V กำลังโตแค่ไหน — มี customer สำคัญคนไหนที่เริ่มเปลี่ยนจาก ARM ไป RISC-V แล้ว
3. SoftBank ถือหุ้นเท่าไหร่และมีแผนขายไหม — ถ้าขายออกมาจะกระทบราคาและ governance อย่างไร
4. v9 adoption อยู่ที่กี่เปอร์เซ็นต์ของ royalty base ตอนนี้ — และถ้า migration เสร็จหมดแล้ว อะไรจะเป็น growth driver ต่อไป
5. "เงื่อนไขที่จะทำให้ผมขายหุ้นนี้ทิ้ง" คืออะไรกันแน่ — ถ้าตอบไม่ได้ แสดงว่ายังไม่พร้อมซื้อ
