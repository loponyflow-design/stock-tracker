# AMZN — Amazon.com
_Brief date: 2026-07-02_

---

## 1. Company snapshot

Amazon มีธุรกิจหลัก 3 ส่วน: e-commerce ในอเมริกาเหนือ (ขายของออนไลน์ + บริการผู้ขายบุคคลที่สาม + โฆษณา), e-commerce ต่างประเทศ และ AWS (บริการ cloud computing ให้องค์กรทั่วโลก) ลูกค้าคือผู้บริโภคทั่วไปกว่า 200 ล้าน Prime member, ผู้ขายบุคคลที่สามกว่า 3 ล้านราย และองค์กร/นักพัฒนาที่ใช้ AWS แม้ AWS จะเป็นแค่ 18% ของรายได้รวม แต่เป็นแหล่งกำไรหลักของบริษัท (53% ของ operating income) เพราะมาร์จิ้นสูงกว่าธุรกิจค้าปลีกมาก

---

## 2. Fundamentals signal

- **รายได้โตสม่ำเสมอ:** $574.8B → $638.0B → $716.9B (CAGR ~11.5%) AWS โตเร็วสุด (+19.8%) และโฆษณาโตแรงสุด (+22.1%)
- **มาร์จิ้นขยายตัวต่อเนื่อง:** operating margin จาก 6.4% (FY2023) เป็น 11.2% (FY2025) ผลจาก mix รายได้ AWS/โฆษณาที่มาร์จิ้นสูงขึ้นเรื่อยๆ
- **AWS แบกกำไรทั้งบริษัท:** AWS margin 35.4% เทียบ North America 6.9% และ International 2.9% — ธุรกิจ retail ทำหน้าที่สร้าง scale/ฐานลูกค้า ส่วน AWS เป็นเครื่องจักรทำกำไรจริง
- **ทุ่มลงทุน AI capex หนักมาก:** capex พุ่ง +58.8% เป็น $131.8B (18.4% ของรายได้) ผู้บริหารประกาศจะใช้ capex ปี 2026 ถึง $200B — FCF ยังบวก ($38.2B) แต่ถูกกดจากการลงทุนหนัก
- **งบดุลแกร่ง เงินสดสุทธิ $57.4B** หนี้ต่ำเทียบสินทรัพย์รวม (8%) — มีกันชนรองรับการลงทุนหนักได้

## 4. Bull case / Bear case

**Bull**
- **Scale economies ใน fulfillment + AWS** — เครือข่าย logistics และ data center ขนาดใหญ่ที่สุดในอุตสาหกรรม ทำให้ต้นทุนต่อหน่วยต่ำกว่าคู่แข่งรายเล็ก และสร้างอุปสรรคให้คู่แข่งใหม่เข้ามาแข่งขันในระดับเดียวกันยากมาก
- **AWS เป็นแหล่งกำไรที่มี switching cost สูง** — องค์กรที่ย้ายระบบมาอยู่บน AWS แล้วเปลี่ยนไปใช้ cloud อื่นต้องใช้ต้นทุนและความเสี่ยงสูง ทำให้รายได้ recurring มั่นคง
- **กระจายแหล่งรายได้หลายทาง** (e-commerce, cloud, โฆษณา, subscription) ลดการพึ่งพาธุรกิจใดธุรกิจหนึ่งมากเกินไป

**Bear**
- **Capex AI สูงกว่ารายได้ AI ที่เกิดขึ้นจริงมาก** — AWS AI revenue run rate $15B เทียบ capex ปีนี้ $200B เป็นช่องว่างใหญ่ ถ้าดีมานด์ AI ไม่โตทันการลงทุน จะกดดัน FCF และ ROIC ในระยะกลาง
- **ธุรกิจ e-commerce หลักมาร์จิ้นบางมาก** (North America 6.9%, International 2.9%) — ไวต่อการแข่งขันด้านราคาและกำลังซื้อผู้บริโภค ถ้าเศรษฐกิจแย่ ผู้บริโภคลดใช้จ่าย margin ส่วนนี้จะถูกกดดันก่อน
- **Guidance Q2 บ่งชี้ margin หดตัว** เป็นสัญญาณว่าการลงทุนหนักเริ่มกดดันกำไรระยะสั้น แม้รายได้ยังโต

---

## 5. Kill conditions

- **ถ้า AWS AI revenue growth ไม่เร่งตัวขึ้นให้ทันกับ capex ที่ทุ่มไป** ต่อเนื่องหลายไตรมาส — เป็นสัญญาณว่าการลงทุนอาจไม่คุ้มทุนตามที่คาด
- **ถ้า operating margin ของ North America/International หดตัวต่อเนื่องจากการแข่งขันด้านราคา** — บ่งชี้ว่าอำนาจต่อรองในธุรกิจ retail หลักเริ่มอ่อนลง
- **ถ้า AWS เริ่มเสียส่วนแบ่งตลาดให้คู่แข่ง (Azure, Google Cloud) อย่างมีนัยสำคัญ** — เพราะ AWS คือแหล่งกำไรหลักของทั้งบริษัท

---

## 6. What to ask before owning it

1. Capex AI $200B ต่อปี คุ้มค่ากับรายได้ AI ที่กำลังเกิดขึ้นจริงหรือยัง และผมให้เวลาบริษัทพิสูจน์ได้กี่ปี?
2. ถ้าเศรษฐกิจสหรัฐฯ ชะลอตัวและผู้บริโภคลดใช้จ่าย ธุรกิจ retail ที่มาร์จิ้นบางอยู่แล้วจะกระทบแค่ไหน?
3. AWS ยังรักษาความเป็นผู้นำตลาด cloud ได้แค่ไหนเมื่อเทียบกับ Microsoft Azure และ Google Cloud ที่ก็ทุ่มลงทุน AI เช่นกัน?
4. ผมเข้าใจโครงสร้างที่ AWS แบกกำไรทั้งบริษัทดีพอไหม ถ้า AWS สะดุดจะกระทบภาพรวมมากแค่ไหน?
5. "เงื่อนไขที่จะทำให้ผมขายทิ้ง" คืออะไร — margin หดตัวต่อเนื่อง, AWS เสียส่วนแบ่งตลาด, หรือ capex ไม่แปลงเป็นรายได้จริงตามที่คาด?
