# Microsoft Corporation 10-K FY2025 (excerpt — investment summary)

## ธุรกิจหลัก

Microsoft พัฒนาและขาย software, cloud services, hardware และ AI solutions ให้กับ individual users, enterprises, governments และ developers ทั่วโลก รายได้หลักมาจาก 3 segments: Microsoft 365 + LinkedIn + Dynamics (Productivity & Business Processes), Azure + server products (Intelligent Cloud) และ Windows + Xbox + Bing (More Personal Computing) บริษัทควบคุม platform ตั้งแต่ OS → middleware → cloud infrastructure → application layer และมี distribution ผ่านทั้ง direct enterprise agreements, OEM partnerships และ cloud marketplace

## Product lines

- **Microsoft 365 Commercial** — subscription bundle (Word, Excel, PowerPoint, Teams, Exchange, SharePoint, OneDrive, Copilot)
- **Microsoft 365 Consumer** — M365 Personal/Family subscriptions; 89M subscribers (+8% YoY)
- **Azure** — cloud computing platform (IaaS, PaaS, serverless, AI/ML); revenue +34% YoY
- **GitHub / Developer tools** — version control, CI/CD, Copilot for code
- **Dynamics 365** — ERP/CRM cloud suite; +19% YoY
- **LinkedIn** — professional network + job listings + recruiting + learning; +9% YoY
- **Windows** — OEM + commercial licensing; +3% YoY
- **Xbox** — console hardware + Game Pass subscription + content/services; +16% YoY
- **Bing / Microsoft Copilot** — search engine + AI chat; search advertising ex-TAC +20% YoY
- **Surface / Devices** — hardware (relatively small share)
- **Server products (on-premise)** — SQL Server, Windows Server, System Center; bundled with hybrid cloud strategy

## Distribution

- **Enterprise (direct + partner)**: ขาย via Enterprise Agreement / Microsoft Customer Agreement ตรงให้ enterprise; ผ่าน distributors, resellers, SIs
- **OEM**: Windows pre-installed บน PC/laptop ผ่าน Lenovo, HP, Dell, Asus และอื่นๆ
- **Cloud Marketplace**: Azure Marketplace, AppSource — channel สำหรับ ISV และ developer tools
- **Retail**: Microsoft Store (online + บางสาขาที่ยังคงอยู่) + third-party retailers สำหรับ Surface, Xbox, M365 consumer
- **Revenue mix**: Service & other (cloud + subscriptions) = 77.3% ของ total revenue ($217.8B); Product (one-time licenses + hardware) = 22.7% ($63.9B)

## การแข่งขัน

Microsoft แข่งในหลาย market พร้อมกัน:
- **Cloud IaaS/PaaS**: AWS (#1) > Azure (#2) > Google Cloud (#3) — Azure ใกล้ขึ้นไปเรื่อยๆ
- **Productivity suite**: Google Workspace — M365 ยังนำในองค์กรขนาดกลาง-ใหญ่
- **AI assistant**: OpenAI ChatGPT (partner ไม่ใช่คู่แข่งโดยตรง), Google Gemini, Anthropic Claude
- **CRM/ERP**: Salesforce (CRM), SAP/Oracle (ERP) vs Dynamics 365
- **Gaming**: Sony PlayStation, Nintendo vs Xbox; Steam vs Xbox Game Pass
- ข้อได้เปรียบ: ecosystem lock-in (Windows → M365 → Azure → Copilot), distribution scale, enterprise trust, OpenAI partnership

## Supply chain — ความเสี่ยง

- **Datacenter build-out**: capex FY2025 = $64.6B (เพิ่มขึ้นจาก $44.5B FY2024) — ต้องใช้ GPU (NVIDIA), networking gear, land, power; construction commitments อีก $32.1B
- **Semiconductor dependencies**: เน้น NVIDIA GPUs สำหรับ AI training/inference; OpenAI มี right of first refusal บน Azure capacity
- **Energy & power**: datacenter ต้องการ power มหาศาล ข้อจำกัด grid เป็นความเสี่ยง
- **Purchase commitments**: $109.9B total (cloud infrastructure, AI chips, network equipment)
- **Geopolitical risk**: Taiwan Strait (TSMC dependency), China revenue exposure, US-China tech export controls

## Seasonality

- **Q2 (Oct-Dec)** สูงสุด: holiday season ดัน Xbox, Surface, consumer M365
- **Q4 (Apr-Jun)** สูงสุดในฝั่ง commercial: enterprise ปิด deal ก่อนสิ้นปีงบประมาณ + Microsoft FY end
- Unearned revenue (deferred) $67.3B ณ สิ้น FY2025 — ส่วนใหญ่ recognize ใน 12 เดือนถัดไป

## Human capital

- พนักงาน FTE: **228,000** (ณ 30 มิ.ย. 2025)
  - สหรัฐฯ: 125,000
  - ต่างประเทศ: 103,000
- มี AI Skilling initiative: บริษัท commit training พนักงาน + 1M+ ผู้ใช้ภายนอกในด้าน AI skills

## IP

- Patent portfolio ขนาดใหญ่ครอบคลุม OS, cloud, AI, productivity software
- Trademark: "Microsoft", "Azure", "Windows", "Xbox", "LinkedIn", "Bing", "Copilot", "Dynamics"
- Copyright: source code ใน Windows, Office, Azure, เกม Xbox studios
- OpenAI: exclusive Azure API provider + revenue sharing agreement + right of first refusal on capacity — เป็น IP ทางกลยุทธ์ที่สำคัญที่สุด

## FY2025 Financial Summary

### Revenue drivers

- **Azure & cloud services (+34%)**: AI workloads + enterprise migration หลัก; Microsoft Cloud total $168.9B (+23%)
- **Dynamics 365 (+19%)**: ERP/CRM migration cloud ต่อเนื่อง
- **Search & news advertising ex-TAC (+20%)**: Bing/Copilot ได้ประโยชน์จาก AI integration
- **Xbox content & services (+16%)**: Game Pass + Activision-Blizzard catalog
- **M365 Commercial cloud (+15%)**: M365 seat growth +6%, Copilot upsell
- **LinkedIn (+9%)**: hiring demand ยังคงอยู่ แม้ market ชะลอ
- **M365 Consumer (+11%)**: subscriber base 89M, Copilot driving upgrade
- **Windows OEM (+3%)**: PC market recovery ช้า

### Gross margin

- **Total gross margin**: $193,893M / $281,724M = **68.8%** (FY2024: 69.8%)
- ลดลง 1pp เพราะ Intelligent Cloud margin กดดัน (datacenter depreciation เพิ่ม): Intelligent Cloud GM ≈ 70% vs Productivity segment GM ≈ 80%+
- Microsoft Cloud gross margin: **69%** — ลดลงจาก 71% ปีก่อน เพราะ AI infra cost เพิ่ม

- G&A ลดลงเพราะ restructuring ที่ผ่านมาเห็นผล

### Liquidity & capital return

- **Cash + ST investments**: $94,565M ($94.6B) — net cash สุทธิ (ไม่มีหนี้สุทธิ) หลังหักหนี้ทั้งหมด
- **LT Debt**: $40,152M; Total debt (including current): ~$43.2B principal $49.2B
- **Operating CF**: $136,162M (+18% YoY)
- **Capex**: $(64,551)M — AI datacenter build-out (+45% YoY จาก $44.5B FY2024)
- **FCF** (Operating CF - Capex): ~$71,611M
- **Share buybacks**: $(18,420)M
- **Dividends**: $(24,082)M
- **Total capital returned**: ~$42,502M
- Construction commitments: $32.1B (ongoing datacenter build)
- Purchase commitments: $109.9B (chips, network, cloud infra)

### Tax

- **Effective tax rate: 18%** (FY2025, FY2024, consistent)
- ต่ำกว่า statutory 21% เพราะ R&D credits, foreign derived intangible income deduction, stock compensation deductions
- **OBBBA (One Big Beautiful Bill Act)** ผ่านวันที่ 4 กรกฎาคม 2025 — บริษัทระบุว่ากำลังประเมินผลกระทบ; 100% bonus depreciation อาจช่วย capex deduction

## Key Risk Factors

### Macro & geopolitical

- **US-China trade / tech controls**: export restrictions บน AI chips และ cloud services กระทบ revenue ใน China + Asia; กฎหมายใหม่อาจบังคับให้ Microsoft ถอน products จากบางตลาด
- **Regulatory scrutiny (global)**: EU Digital Markets Act กำหนดให้ Microsoft เปิด interoperability (Teams, Edge, Windows) — compliance cost + potential product changes
- **Currency / FX**: revenue สร้างใน >100 currencies; strong USD กด translated revenue (FX headwind ≈ -1-2% ต่อปี)
- **Economic downturn**: enterprise ชะลอ cloud migration + hiring ลด LinkedIn revenue; consumer M365 churn เพิ่ม
- **AI regulation**: EU AI Act, US executive orders — อาจ restrict deployment ของ Azure AI + Copilot ในบางกรณี

### Regulatory & legal

- **Antitrust**: DOJ/FTC ตรวจสอบ Microsoft ต่อเนื่อง (Teams bundling, cloud licensing); EU จบ Teams bundling issue แล้วแต่ยังมี cloud licensing investigation
- **Activision-Blizzard**: หลัง merger อยู่ระหว่างตรวจสอบ competition ใน gaming market
- **Data privacy**: GDPR (EU), CCPA (California), Australia, India data localization — compliance cost + product modification
- **Cybersecurity liability**: high-profile breaches (SolarWinds supply chain, Chinese state actors เจาะ Exchange Online ปี 2023) ทำให้ SEC + Congress สอบสวน Microsoft security practices; CSRB report ระบุ "cascade of security failures"
- **OpenAI governance**: Microsoft เป็น investor + exclusive partner — ถ้า OpenAI มีปัญหา governance/safety Microsoft จะโดน reputational risk

### Business risks

- **Capex execution risk**: $64.6B capex FY2025 + $32.1B construction commitments = ต้องการ AI demand จริงเพื่อ justify ROI; ถ้า AI adoption ช้ากว่าคาด return on datacenter อาจต่ำกว่า
- **Competition AI**: Google ลงทุน Gemini integration ใน Workspace โดยตรง; Amazon Bedrock + AWS เป็น default สำหรับ AI ใน AWS ecosystem; OpenAI อาจสร้าง competing products
- **Cybersecurity threats to self**: Microsoft เป็น target ใหญ่ที่สุดในโลก; breach ใน Azure หรือ M365 จะกระทบ trust ของลูกค้า enterprise
- **Talent retention**: AI talent แข่งกันจาก Google, Meta, OpenAI, Anthropic; equity incentive ต้องแข่งขัน
- **Gaming**: Xbox console hardware share ลดลง เทียบ PlayStation; Game Pass subscriber growth ชะลอ
